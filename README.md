# PEPlife2: Repository of Peptide & Protein Half-life Data

Welcome to the official repository for **PEPlife2**, an updated and manually curated database of experimentally validated peptide and protein half-life data. This resource is designed to support researchers working in peptide therapeutics, computational biology, bioinformatics, drug delivery, and protein engineering.

Web Server: https://webs.iiitd.edu.in/raghava/peplife2/  
API Access: https://webs.iiitd.edu.in/raghava/peplife2/api/rest.html

# About the Database

PEPlife2 is an updated version of the original PEPlife database published in 2016. The database compiles experimentally validated information on peptide and protein half-life collected from research articles and patents published between 2016 and 2024.

The platform integrates detailed information related to:

* Peptide and protein sequences
* Half-life values
* Experimental assay methods
* Chemical and physical modifications
* Structural information
* Biological activity
* Routes of administration
* SMILES representations

The database was manually curated from:

* PubMed
* The Lens patents database
* Published literature
* Experimental reports

---

# Key Features

## Large Curated Dataset

* 4,412 total entries
* 1,781 peptides
* 402 proteins
* Data curated from:
  * 449 research articles
  * 18 patents
* Information from 105 unique assays

---

## Comprehensive Annotations

Each entry includes:

* Peptide/protein sequence
* Name and length
* Half-life value
* Assay type
* Chirality
* Linear or cyclic structure
* N-terminal modification
* C-terminal modification
* Chemical modification
* Physical modification
* Biological activity
* Organism and media
* Route of administration
* Patent ID
* SMILES notation

---

## Structural Information

Structural annotations were obtained using:

* Protein Data Bank (PDB)
* PEPstrMOD
* I-TASSER
* DSSP secondary structure analysis

The database provides experimentally validated and predicted structural information for peptides and proteins.

# Important Modifications Included

PEPlife2 contains information on several half-life enhancement strategies such as:

* PEGylation
* Fc-conjugation
* Cyclization
* Amidation
* Acetylation
* D-amino acid substitutions
* Non-natural amino acid incorporation
* Fatty acid conjugation
* Nanoparticle-based physical modifications
* Hydrogel-based modifications

These modifications help improve peptide stability and therapeutic efficiency.
